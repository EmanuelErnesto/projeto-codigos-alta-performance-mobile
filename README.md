# Moeda Converter - Aplicação de Conversão de Moedas

Este é um projeto de aplicação de conversão de moedas com funcionalidades de consulta em tempo real, que utiliza Ionic Framework. A aplicação permite a conversão entre diferentes moedas e visualiza a taxa de câmbio de moedas em tempo real, além de outras funcionalidades como a atualização de valores e visualização de moedas em alta e em baixa.

## Requisitos

Antes de começar, verifique se você tem os seguintes pré-requisitos instalados no seu sistema:

- **Node.js** (Recomendado a versão 16.x ou superior)
- **npm** (gerenciador de pacotes do Node.js, que já vem instalado com o Node.js)
- **Ionic CLI** (Ferramenta de linha de comando do Ionic)
- **Angular** (Usado no Ionic para desenvolvimento)

### Instalação do Node.js e npm
Caso ainda não tenha o Node.js e o npm, siga os passos para instalá-los:

1. Vá até [nodejs.org](https://nodejs.org/) e baixe a versão mais recente LTS.
2. Após a instalação, verifique se foi bem sucedido rodando no terminal:
   ```bash
   node -v
   npm -v
    ```

## Tecnologias Utilizadas

- **Ionic Framework**: Uma plataforma para o desenvolvimento de aplicativos móveis e web híbridos.
- **Angular**: Framework JavaScript utilizado para construir a aplicação.
- **Ionic Components**: UI altamente personalizável para a construção de interfaces responsivas.
- **API de Moedas**: Conexão com uma API para obter taxas de câmbio em tempo real (a ser definida dependendo da sua implementação).
- **CSS/SCSS**: Estilização do projeto com foco em uma interface limpa e moderna.

## Funcionalidades

- **Conversão de Moedas**: O usuário pode selecionar a moeda de origem e destino, além de inserir o valor a ser convertido. O valor convertido será mostrado em tempo real com a taxa de câmbio atual.

- **Moedas em Tempo Real**: O app permite consultar as taxas de câmbio em tempo real para diferentes moedas, além de visualizar as moedas em alta e em baixa.

- **Reset de Senha e Login**: Funcionalidades de login e recuperação de senha para os usuários que desejam personalizar suas preferências.

- **Interface Responsiva**: A aplicação é totalmente responsiva e adaptada para dispositivos móveis, garantindo uma experiência fluida tanto em smartphones quanto em tablets.

## Como Rodar o Projeto

### Passo 1: Clonar o Repositório
Comece clonando o repositório para sua máquina local:

```bash
git clone https://github.com/EmanuelErnesto/projeto-codigos-alta-performance-mobile.git
cd projeto-codigos-alta-performance-mobile
```

### Passo 2: Instalar Dependências
Dentro do diretório do projeto, execute o seguinte comando para instalar todas as dependências necessárias:

```bash
npm install
```

### Passo 3: Rodar a Aplicação em um Servidor Local
Após instalar as dependências, você pode iniciar o servidor de desenvolvimento:

```bash
npm start
```
